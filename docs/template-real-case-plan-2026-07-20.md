# 18 套真实模板：商品与提示词清单

更新时间：2026-07-20

## 生成方法

- 图像模型：APIMart `gpt-image-2`
- 生成方式：图生图；每套案例传入 1 张真实商品母图，并在提示词中锁定商品本体
- 输出：2K，比例跟随模板
- 商品一致性：15 套严格保留商品本体；3 套 Brand Kit 只允许重做品牌图形层，不允许改变产品结构
- 完整、可直接复制的最终英文提示词：在项目根目录运行 `npm run templates:real:prompts`
- 可执行数据源：`template-real-cases.js`

最终提示词统一由三段组成：

1. 商品锁定约束：轮廓、比例、结构、材质、颜色、包装几何和可见细节以参考图为唯一事实来源。
2. 模板美术方向：场景、光线、构图、比例、留白和渠道用途。
3. 最终参考图指令：解决通用模板描述与实际商品不一致的问题，并覆盖冲突项。
