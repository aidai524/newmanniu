# 通知中心内容模板

版本：2026.07.21-2  
对应实现：`notification-catalog.js`

本文档定义通知中心的分类、模板文案、动态变量与操作入口。通知实例由“模板 ID + 变量数据”生成，前端不应单独硬编码标题或正文。

## 一、通知分类

| 分类键 | 分类名称 | 收录范围 |
| --- | --- | --- |
| `task` | 创作任务 | 图片、视频及其他创作任务的完成、失败和导出结果 |
| `asset` | 资产与协作 | 资产分享、成员留言和协作动态 |
| `account` | 账户设置 | 账户头像、登录密码和基础资料的变更 |
| `billing` | 会员与额度 | 套餐到期、积分余额和订单支付结果 |
| `system` | 产品与系统 | 功能上线、服务维护和规则更新 |

## 二、创作任务

### 1. 任务完成

- 模板 ID：`task.completed`
- 事件标签：任务完成
- 提醒级别：成功
- 标题：你的「{{workName}}」已生成完成
- 正文：共生成{{count}}{{assetUnit}}，已自动保存至「我的资产」，现在可以预览或下载。
- 操作按钮：查看结果
- 目标页面：我的资产
- 动态变量：`workName` 作品名称；`count` 结果数量；`assetUnit` 结果单位，例如“张图片”或“段视频”

### 2. 任务未完成

- 模板 ID：`task.failed`
- 事件标签：需要处理
- 提醒级别：重要
- 标题：「{{workName}}」生成未完成
- 正文：{{reason}}。本次未扣除生成额度，你可以调整素材后重新尝试。
- 操作按钮：返回调整
- 目标页面：快速生成
- 动态变量：`workName` 作品名称；`reason` 未完成原因

### 3. 导出完成

- 模板 ID：`task.exported`
- 事件标签：导出完成
- 提醒级别：信息
- 标题：「{{workName}}」已完成导出
- 正文：{{format}}文件已经生成，并保留在「我的资产」中供你再次下载。
- 操作按钮：查看文件
- 目标页面：我的资产
- 动态变量：`workName` 作品名称；`format` 导出格式，例如“PNG”或“MP4”

## 三、资产与协作

### 1. 收到资产分享

- 模板 ID：`asset.shared`
- 事件标签：收到分享
- 提醒级别：信息
- 标题：{{sender}}向你分享了「{{assetName}}」
- 正文：你已获得“{{permission}}”权限，可以打开资产查看内容和相关版本。
- 操作按钮：打开资产
- 目标页面：我的资产
- 动态变量：`sender` 分享人；`assetName` 资产名称；`permission` 权限，例如“查看与评论”

### 2. 收到协作留言

- 模板 ID：`asset.commented`
- 事件标签：协作动态
- 提醒级别：信息
- 标题：{{sender}}在「{{assetName}}」中留言
- 正文：“{{comment}}”打开资产可查看完整上下文并继续处理。
- 操作按钮：查看留言
- 目标页面：我的资产
- 动态变量：`sender` 留言人；`assetName` 资产名称；`comment` 留言摘要

## 四、账户设置

### 1. 头像变更

- 模板 ID：`account.avatar_changed`
- 事件标签：头像变更
- 提醒级别：信息
- 标题：你的账户头像已更新
- 正文：账户头像已于{{changedAt}}更新。若非本人操作，请检查账户设置并及时修改密码。
- 操作按钮：查看账户设置
- 目标页面：账户设置
- 动态变量：`changedAt` 变更时间

### 2. 密码变更

- 模板 ID：`account.password_changed`
- 事件标签：密码变更
- 提醒级别：提醒
- 标题：你的登录密码已修改
- 正文：登录密码已于{{changedAt}}修改。为了账户安全，其他已登录设备可能需要重新验证。
- 操作按钮：查看账户设置
- 目标页面：账户设置
- 动态变量：`changedAt` 变更时间

### 3. 基础资料变更

- 模板 ID：`account.profile_changed`
- 事件标签：资料变更
- 提醒级别：信息
- 标题：你的账户基础资料已更新
- 正文：{{fields}}已于{{changedAt}}更新。你可以进入账户设置查看最新资料。
- 操作按钮：查看账户设置
- 目标页面：账户设置
- 动态变量：`fields` 变更字段摘要，例如“昵称和企业名称”；`changedAt` 变更时间

## 五、会员与额度

### 1. 会员即将到期

- 模板 ID：`billing.plan_expiring`
- 事件标签：会员到期
- 提醒级别：提醒
- 标题：{{planName}}将在{{days}}天后到期
- 正文：续费后可继续使用当前积分、{{storage}} 资产空间与无水印导出能力。
- 操作按钮：查看会员
- 目标页面：我的权益
- 动态变量：`planName` 套餐名称；`days` 剩余天数；`storage` 资产空间额度

### 2. 生成积分不足

- 模板 ID：`billing.credits_low`
- 事件标签：额度提醒
- 提醒级别：提醒
- 标题：本周期生成积分仅剩{{credits}}
- 正文：积分将在{{resetDate}}重置。你可以查看明细或按需补充积分。
- 操作按钮：查看积分
- 目标页面：积分明细
- 动态变量：`credits` 剩余积分；`resetDate` 重置日期

### 3. 支付成功

- 模板 ID：`billing.payment_succeeded`
- 事件标签：支付成功
- 提醒级别：成功
- 标题：{{orderName}}支付成功
- 正文：实付{{amount}}，相关会员权益或积分已经到账。
- 操作按钮：查看订单
- 目标页面：我的订单
- 动态变量：`orderName` 订单名称；`amount` 实付金额

## 六、产品与系统

### 1. 功能上新

- 模板 ID：`system.feature_release`
- 事件标签：功能上新
- 提醒级别：信息
- 标题：{{featureName}}现已上线
- 正文：{{summary}}你可以立即打开相关功能查看完整内容。
- 操作按钮：立即体验
- 目标页面：模板中心或对应功能页面
- 动态变量：`featureName` 功能名称；`summary` 功能摘要

### 2. 系统维护

- 模板 ID：`system.maintenance`
- 事件标签：服务维护
- 提醒级别：提醒
- 标题：{{date}}将进行系统维护
- 正文：预计维护{{duration}}，期间{{impact}}，已经开始的任务不会丢失。
- 操作按钮：查看帮助
- 目标页面：反馈帮助
- 动态变量：`date` 维护开始时间；`duration` 预计时长；`impact` 影响范围

### 3. 规则更新

- 模板 ID：`system.policy_updated`
- 事件标签：规则更新
- 提醒级别：信息
- 标题：{{policyName}}已更新
- 正文：新版本将于 {{effectiveDate}} 生效，请在使用相关功能前了解主要变化。
- 操作按钮：查看说明
- 目标页面：反馈帮助
- 动态变量：`policyName` 规则或协议名称；`effectiveDate` 生效日期

## 七、使用约定

1. 标题用于说明“发生了什么”，正文用于说明结果、影响及用户下一步。
2. 标题和正文中的 `{{variable}}` 必须在创建通知时提供，不应向用户展示未替换变量。
3. 时间、金额、积分和数量由业务层完成格式化后传入模板。
4. “重要”通知仅用于需要用户处理的失败或高风险结果，避免将普通动态标记为重要。
5. 操作按钮应跳转到能够直接处理该通知的页面，避免只返回工作台首页。
